# SubwayProject
* 2022.09.01 ~ 2022.12.15
* 명지대학교 팀프로젝트1

## 프로젝트 소개
* 가상의 지하철 노선도와 역들의 정보가 주어졌을때 사용자의 요구사항에 맞는 정보를 출력하는 에플리케이션을 개발한 프로젝트 입니다.
* 최소환승, 최소거리, 최단시간, 최소비용의 값을 계산하여 원하는 정보를 제공합니다.
* 각 역의 편의시설을 데이터베이스에 저장하여 입력받은 역값에 알맞게 출력합니다.
